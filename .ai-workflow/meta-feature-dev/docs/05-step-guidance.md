# 分步指导文档

## 项目信息
- 项目名称: interview-tiger
- 需求描述: 修复语音识别延迟（添加手动结束按钮）+ UI整体翻新
- 创建时间: 2026-07-02
- 状态: 已完成

---

## 实施步骤

### S1: 修改 useSpeech.ts 新增 forceStop 方法

**目标:** 在语音识别组合式函数中添加强制停止功能

**步骤:**
1. 打开 `frontend/src/composables/useSpeech.ts`
2. 在现有方法之后添加 `forceStop()` 方法
3. 实现逻辑:
   - 调用 `recognition.stop()`
   - 合并 `finalTranscript` 和 `interimTranscript`
   - 触发 `onResult` 回调
   - 重置状态

**风险点:**
- 确保在停止前保存所有临时识别结果
- 处理未开始识别时的边界情况

**验证方法:**
- 启动开发服务器
- 打开浏览器测试语音识别
- 点击完成按钮，确认识别文本立即提交

---

### S2: 修改 InterviewPage.vue 添加完成按钮

**目标:** 在语音识别界面添加手动完成按钮

**步骤:**
1. 打开 `frontend/src/components/InterviewPage.vue`
2. 在录音按钮附近添加"完成"按钮
3. 设置按钮显示条件: `v-if="isListening"`
4. 绑定点击事件: `@click="forceStop"`
5. 添加按钮样式

**风险点:**
- 确保按钮状态与识别状态同步
- 避免按钮重复点击

**验证方法:**
- 测试按钮在识别开始时显示
- 测试按钮在识别结束后隐藏
- 测试点击按钮能立即停止识别

---
