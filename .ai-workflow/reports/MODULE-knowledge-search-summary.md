# 知识库检索模块 - 模块摘要

## 📌 一句话概括

实现火山引擎知识库的检索服务，支持SignerV4签名认证，从知识库中检索与面试问题相关的用户简历/项目经历信息。

---

## 👤 面向产品/业务人员

### 这个模块做了什么？

知识库检索模块是面试虎的"记忆库"，负责从用户配置的知识库中查找与面试问题相关的信息。当面试官提出问题时，系统会在知识库中搜索相关的简历内容、项目经历等信息，然后把这些信息提供给大模型，让大模型生成的回答更加个性化，更贴合用户的真实经历。

### 用户可以做什么？

| 用户故事 | 操作路径 | 完成状态 |
|---------|---------|----------|
| 配置知识库 | 在设置中输入知识库ID和API Key | ✅ |
| 检索相关信息 | 提出问题后自动检索 | ✅ |
| 获得个性化回答 | 基于知识库内容生成回答 | ✅ |
| 无知识库时降级 | 自动使用通用模式 | ✅ |

### 涉及几个页面？

此模块为后端服务，通过API调用，前端配置页面输入参数。

## 🔧 面向技术负责人

### 涉及接口清单

| 方法 | 端点 | 说明 |
|------|------|------|
| POST | /api/search | 知识库检索 |
| POST | /api/question | 完整问答流程（包含知识库检索） |

### 关键数据结构

```
KnowledgeService (服务类)
├── search_knowledge(query, kb_id, kb_api_key, **kwargs)
├── get_relevant_knowledge(response)
├── _sign_request(ak, sk, method, url, body)
└── _parse_api_key(kb_api_key)

KnowledgeItem
├── content: string       # 内容
├── score: float          # 相关性分数
├── rerank_score: float   # 重排序分数
├── source: string        # 来源
└── title: string         # 标题
```

### 关键架构决策

| ADR编号 | 决策内容 | 影响面 |
|---------|---------|--------|
| ADR-009 | 选择火山引擎RAG作为知识库服务 | 知识库检索模块 |

### 风险与注意事项

| 风险项 | 级别 | 说明 | 缓解措施 |
|--------|------|------|----------|
| 认证失败 | 🟡中 | API Key格式错误或过期 | 添加详细错误提示和格式校验 |
| 检索结果为空 | 🟡中 | 知识库中无相关内容 | 自动降级为通用模式 |
| 检索延迟 | 🟡中 | 网络延迟可能影响体验 | 添加超时控制和加载状态 |
