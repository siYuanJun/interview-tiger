## 二、业务解读

### 2.1 需求含义解读
用户希望优化面试页面中语音输入状态的视觉反馈：
1. **图标替换**：将当前使用的 Lucide Mic 图标替换为自定义 SVG 语音输入图标，与项目整体 SVG 主题风格保持一致
2. **背景简化**：移除当前的模糊渐变背景效果（`absolute inset-0 bg-gradient-to-br from-primary/30 to-secondary/30 rounded-3xl blur-2xl`），该效果过于厚重，影响整体视觉
3. **动态反馈**：当系统正在监听/识别用户语音时，图标需要有动态变化效果，直观反馈语音输入状态

### 2.2 使用场景
- 面试页面初始状态（等待开始）
- 语音识别监听中（用户正在说话）
- 语音识别处理中（识别结果生成中）

### 3.3 技术实现方案

#### 方案1：创建动态语音波形SVG组件

**组件设计**：
- 创建 `VoiceWave.vue` 组件
- SVG 包含麦克风图标 + 动态波形动画
- 根据语音状态显示不同动画效果

**动画效果设计**：
| 状态 | 动画效果 |
|------|---------|
| idle | 静态麦克风图标，无波形 |
| starting | 缓慢脉冲发光效果 |
| listening | 动态声波波形，随语音强度变化 |
| recognizing | 旋转/扫描效果 |
| error | 红色闪烁警告 |

**技术实现**：
- 使用 SVG 路径绘制声波波形
- 使用 CSS animation 或 JS 控制波形高度变化
- 通过 props 接收语音状态，动态切换动画

#### 方案2：使用CSS动画实现动态效果

**设计**：
- 保留 Lucide Mic 图标
- 添加多层圆形波纹动画模拟声波扩散
- 根据语音状态切换波纹颜色和速度

**动画效果**：
- `listening`：多层波纹向外扩散
- `recognizing`：波纹加速扩散 + 颜色变化
