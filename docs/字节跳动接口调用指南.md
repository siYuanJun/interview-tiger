## 2. 大模型接口（豆包 / Ark）

### 2.1 接口说明

- **兼容协议**：OpenAI Chat Completions 格式
- **接口地址**：`https://ark.cn-beijing.volces.com/api/v3/chat/completions`
- **认证方式**：HTTP Header `Authorization: Bearer {API_KEY}`
- **模型名称**：`doubao-seed-2-0-mini-260215`（推荐，备选：`doubao-1-5-pro-32k-250115`）
- **超时建议**：30 秒

### 2.2 如何获取 API Key

**获取步骤：**

1. 登录 [火山引擎控制台](https://console.volcengine.com/)
2. 进入 **火山方舟** → **模型广场**
3. 点击右上角头像 → **API Key**
4. 点击 **创建 API Key**
5. 复制生成的 API Key（格式：`YOUR_ARK_API_KEY`）

> ⚠️ **注意**：API Key 仅在创建时显示一次，请妥善保存。

### 2.3 请求格式

**URL**：`POST https://ark.cn-beijing.volces.com/api/v3/chat/completions`

**Headers**：
```json
{
  "Content-Type": "application/json",
  "Authorization": "Bearer YOUR_ARK_API_KEY"
}
```

**Body（通用对话）**：
```json
{
  "model": "doubao-seed-2-0-mini-260215",
  "messages": [
    {
      "role": "user",
      "content": "你的问题或提示词"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 500,
  "stream": false
}
```

**Body（流式对话，stream = true）**：
```json
{
  "model": "doubao-seed-2-0-mini-260215",
  "messages": [
    {
      "role": "user",
      "content": "你的问题或提示词"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 500,
  "stream": true
}
```

### 2.4 响应格式

**非流式响应**（`stream: false`）：
```json
{
  "choices": [
    {
      "message": {
        "content": "模型生成的回答文本..."
      }
    }
  ]
}
```

**流式响应**（`stream: true`）— SSE 格式：
```
data: {"choices":[{"delta":{"content":"你"}}]}

data: {"choices":[{"delta":{"content":"好"}}]}

...

data: [DONE]
```

解析规则：
- 每行以 `data: ` 开头
- 内容为 JSON，从 `choices[0].delta.content` 取文本增量
- 遇到 `data: [DONE]` 表示结束

## 3. 知识库接口（火山引擎知识库）

### 3.1 接口说明

- **接口地址**：`https://api-knowledgebase.mlp.cn-beijing.volces.com/api/knowledge/collection/search_knowledge`
- **认证方式**：HTTP Header `Authorization: Bearer {VIKING_API_KEY}`
- **超时建议**：30 秒

### 3.2 如何获取 VIKING_API_KEY

**获取步骤：**

1. 登录 [火山引擎控制台](https://console.volcengine.com/)
2. 进入 **向量数据库 VikingDB** → **知识库**
3. 点击右上角头像 → **用户管理**
4. 复制 **API Key**（格式：`YOUR_VIKING_API_KEY`）

> ⚠️ **注意**：API Key 仅在创建时显示一次，请妥善保存。

### 3.3 如何获取知识库 ID（resource_id）

**获取步骤：**

1. 登录 [火山引擎控制台](https://console.volcengine.com/)
2. 进入 **向量数据库 VikingDB** → **知识库**
3. 在知识库列表中找到目标知识库
4. **知识库名称**（如 `siyuan_jianli`）用于 `name` 参数
5. **知识库 ID**（如 `YOUR_KNOWLEDGEBASE_ID`）用于 `resource_id` 参数

> ✅ **推荐**：使用 `resource_id` 参数，比 `name` 更稳定可靠。

### 3.4 配置参数汇总

| 参数 | 值 | 获取位置 |
|---|---|---|
| `KB_API_KEY` | VIKING_API_KEY | 知识库控制台 → 用户管理 |
| `KB_ID` | 知识库 ID（`kb-xxx`） | 知识库列表 → 目标知识库 |
| `KB_PROJECT` | `default` | 默认值，无需修改 |

### 3.5 请求格式

**URL**：`POST https://api-knowledgebase.mlp.cn-beijing.volces.com/api/knowledge/collection/search_knowledge`

**Headers**：
```json
{
  "Content-Type": "application/json",
  "Authorization": "Bearer YOUR_VIKING_API_KEY"
}
```

**Body（使用 resource_id）**：
```json
{
  "resource_id": "YOUR_KNOWLEDGEBASE_ID",
  "project": "default",
  "query": "用户的问题",
  "limit": 3,
  "post_processing": {
    "rerank_switch": true,
    "retrieve_count": 25
  }
}
```

**Body（使用 name）**：
```json
{
  "name": "siyuan_jianli",
  "project": "default",
  "query": "用户的问题",
  "limit": 3,
  "post_processing": {
    "rerank_switch": true,
    "retrieve_count": 25
  }
}
```

### 3.6 参数说明

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `resource_id` | string | 二选一 | 知识库 ID（`kb-xxx`），与 `name` 二选一 |
| `name` | string | 二选一 | 知识库名称，与 `resource_id` 二选一 |
| `project` | string | 是 | 项目名称（默认 `default`） |
| `query` | string | 是 | 检索查询文本 |
| `limit` | int | 是 | 返回结果数量上限 |
| `post_processing.rerank_switch` | bool | 否 | 是否开启重排序 |
| `post_processing.retrieve_count` | int | 否 | 初检数量（rerank 前召回条数） |

### 3.7 响应格式

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "collection_name": "siyuan_jianli",
    "count": 3,
    "result_list": [
      {
        "content": "匹配到的知识文本内容...",
        "score": 0.106,
        "rerank_score": 0.92,
        "id": "切片ID",
        "doc_info": {
          "doc_name": "文档名称.md"
        }
      }
    ]
  }
}
```

### 3.8 响应处理逻辑

```python
# 1. 检查 code == 0 表示成功
# 2. 遍历 data.result_list
# 3. 优先使用 rerank_score，无则用 score
# 4. 过滤 score < 0.05 的结果
# 5. 最多取 3 条
# 6. 用换行符拼接 content 字段
```

### 3.9 常见错误码

| 错误码 | HTTP状态码 | 说明 | 解决方案 |
|---|---|---|---|
| `1000001` | 400 | invalid api key | 检查 API Key 是否正确 |
| `1000001` | 401 | unauthorized | 检查 API Key 是否过期或被禁用 |
| `1000001` | 403 | check sign error | 使用了错误的认证方式，应使用 Bearer Token |
| `1000005` | 400 | collection not exist | 检查知识库 ID 或名称是否正确 |
| `1000029` | 429 | QPS 超过上限 | 降低调用频率或升级旗舰版 |

### 5.3 脚本二：调用大模型（流式 SSE）

```python
#!/usr/bin/env python3
"""
调用火山引擎豆包大模型 - 流式对话 (SSE)
"""

import requests
import json

API_URL = "https://ark.cn-beijing.volces.com/api/v3/chat/completions"
API_KEY = "YOUR_ARK_API_KEY"
MODEL = "doubao-seed-2-0-mini-260215"

headers = {
    "Content-Type": "application/json",
    "Authorization": f"Bearer {API_KEY}"
}

payload = {
    "model": MODEL,
    "messages": [
        {"role": "user", "content": "介绍一下北京故宫"}
    ],
    "temperature": 0.7,
    "max_tokens": 500,
    "stream": True
}

response = requests.post(API_URL, headers=headers, json=payload, timeout=30, stream=True)

if response.status_code == 200:
    print("回答: ", end="", flush=True)
    for chunk in response.iter_lines():
        if not chunk:
            continue
        chunk_str = chunk.decode('utf-8')
        if chunk_str.startswith('data: '):
            chunk_str = chunk_str[6:]
        if chunk_str == '[DONE]':
            break
        try:
            chunk_data = json.loads(chunk_str)
            if 'choices' in chunk_data and chunk_data['choices']:
                delta = chunk_data['choices'][0].get('delta', {})
                content = delta.get('content', '')
                if content:
                    print(content, end="", flush=True)
        except json.JSONDecodeError:
            pass
    print()
else:
    print(f"失败: {response.status_code} - {response.text}")
```

### 5.4 脚本三：调用知识库检索

```python
#!/usr/bin/env python3
"""
调用火山引擎知识库检索接口（使用 Bearer Token 认证）
依赖: pip install requests
"""

import requests
import json

API_URL = "https://api-knowledgebase.mlp.cn-beijing.volces.com/api/knowledge/collection/search_knowledge"
API_KEY = "YOUR_VIKING_API_KEY"
KB_ID = "YOUR_KNOWLEDGEBASE_ID"
PROJECT = "default"


def search_knowledge(query):
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {API_KEY}"
    }

    payload = {
        "resource_id": KB_ID,
        "project": PROJECT,
        "query": query,
        "limit": 3,
        "post_processing": {
            "rerank_switch": True,
            "retrieve_count": 25
        }
    }

    response = requests.post(API_URL, headers=headers, json=payload, timeout=30)
    return response.json() if response.status_code == 200 else {}


def get_relevant_knowledge(query):
    result = search_knowledge(query)

    knowledge = []
    if result.get('code') == 0 and 'data' in result and 'result_list' in result['data']:
        for item in result['data']['result_list']:
            if 'content' not in item:
                continue
            score = item.get('score', 1.0)
            rerank_score = item.get('rerank_score', None)
            if rerank_score is not None:
                if float(rerank_score) < 0.3:
                    continue
            else:
                if float(score) < 0.05:
                    continue
            knowledge.append(item['content'])
            if len(knowledge) >= 3:
                break

    return '\n'.join(knowledge)


if __name__ == '__main__':
    print("=== 火山引擎知识库检索测试 ===\n")

    test_queries = [
        "你叫什么",
        "介绍一下你自己",
    ]

    for q in test_queries:
        print(f"查询: {q}")
        result = search_knowledge(q)
        if result and result.get('code') == 0:
            print(f"状态: 成功")
            print(f"命中数量: {result['data']['count']}")
            knowledge = get_relevant_knowledge(q)
            print(f"相关知识 ({len(knowledge)} 字):")
            print(f"{knowledge[:300]}...")
        else:
            print(f"状态: 失败")
            print(f"错误: {result.get('message', 'Unknown')}")
        print("-" * 50)
```

### 5.5 脚本四：完整 RAG 流程（知识检索 → 生成回答）

```python
#!/usr/bin/env python3
"""
完整 RAG 流程示例：
  1. 知识库检索（获取相关知识）
  2. 大模型生成回答（结合知识）

依赖: pip install requests
"""

import requests
import json

# ==================== 配置 ====================

# 大模型配置
LLM_API_URL = "https://ark.cn-beijing.volces.com/api/v3/chat/completions"
LLM_API_KEY = "YOUR_ARK_API_KEY"
LLM_MODEL = "doubao-seed-2-0-mini-260215"

# 知识库配置
KB_API_URL = "https://api-knowledgebase.mlp.cn-beijing.volces.com/api/knowledge/collection/search_knowledge"
KB_API_KEY = "YOUR_VIKING_API_KEY"
KB_ID = "YOUR_KNOWLEDGEBASE_ID"
KB_PROJECT = "default"

SYSTEM_PROMPT = "你是一个专业、友好的智能面试助手。请根据提供的知识库内容回答用户的问题。"


# ==================== 工具函数 ====================

def call_llm(messages, temperature=0.7, max_tokens=500):
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {LLM_API_KEY}"
    }
    payload = {
        "model": LLM_MODEL,
        "messages": messages,
        "temperature": temperature,
        "max_tokens": max_tokens,
        "stream": False
    }
    response = requests.post(LLM_API_URL, headers=headers, json=payload, timeout=30)
    if response.status_code == 200:
        result = response.json()
        return result["choices"][0]["message"]["content"]
    return None


def search_knowledge(query):
    headers = {
        "Content-Type": "application/json",
        "Authorization": f"Bearer {KB_API_KEY}"
    }

    payload = {
        "resource_id": KB_ID,
        "project": KB_PROJECT,
        "query": query,
        "limit": 3,
        "post_processing": {"rerank_switch": True, "retrieve_count": 25}
    }

    response = requests.post(KB_API_URL, headers=headers, json=payload, timeout=30)
    if response.status_code != 200:
        return ""

    result = response.json()
    knowledge = []
    if result.get('code') == 0 and "data" in result and "result_list" in result["data"]:
        for item in result["data"]["result_list"]:
            if "content" not in item:
                continue
            score = item.get("score", 1.0)
            rerank_score = item.get("rerank_score", None)
            if rerank_score is not None:
                if float(rerank_score) < 0.3:
                    continue
            else:
                if float(score) < 0.05:
                    continue
            knowledge.append(item["content"])
            if len(knowledge) >= 3:
                break
    return "\n".join(knowledge)


def rag_chat(user_question):
    print(f"[步骤1] 用户问题: {user_question}")

    knowledge = search_knowledge(user_question)
    print(f"[步骤2] 检索到知识: {'有' if knowledge else '无'}")
    if knowledge:
        print(f"        知识长度: {len(knowledge)} 字")

    prompt = f"{SYSTEM_PROMPT}\n\n"
    if knowledge:
        prompt += f"参考知识：\n{knowledge}\n\n"
    prompt += f"用户的问题：\n{user_question}\n\n"
    prompt += "请根据以上信息，生成一个准确、专业、友好的回答。"

    answer = call_llm(messages=[{"role": "user", "content": prompt}])
    print(f"[步骤3] 生成回答完成\n")
    return answer


# ==================== 测试 ====================
if __name__ == '__main__':
    questions = [
        "你叫什么？",
        "介绍一下你的项目经验",
    ]
    for q in questions:
        print(f"{'='*60}")
        print(f"用户: {q}")
        answer = rag_chat(q)
        print(f"AI: {answer}")
```
