# 首页 UI 美化计划

## 🎨 设计风格定位

### 整体风格：科技感 + 玻璃拟态
- **背景**：深色渐变背景，多层光晕装饰
- **卡片**：玻璃拟态效果（半透明 + 模糊 + 边框发光）
- **按钮**：渐变填充 + 发光效果 + 悬停缩放
- **Logo**：自定义 SVG 老虎图标，金色渐变 + 发光
- **文字**：渐变文字效果，清晰的信息层级

### 配色方案
| 用途 | 颜色 | 说明 |
|------|------|------|
| 主背景 | #0F0F23 | 深紫色背景 |
| 卡片背景 | rgba(30, 27, 75, 0.6) | 半透明玻璃感 |
| 边框 | rgba(123, 58, 237, 0.3) | 紫色边框 |
| 主渐变 | #60a5fa → #c084fc → #f472b6 | 蓝紫粉渐变 |
| Logo | #fbbf24 | 金色老虎图标 |

### 布局细节
- **Logo 区域**：居中，距离顶部约 100px
- **标题**：大字号（56px），渐变文字效果
- **副标题**：中等字号（16px），淡紫色文字
- **功能卡片**：三列网格布局，间距 24px
- **按钮**：宽度 200px，高度 56px，圆角 16px

## 🎯 设计亮点

### 1. Logo 设计
- 使用 hero.svg 中的老虎图标
- 金色渐变填充，配合发光效果
- 浮动动画（上下浮动）

### 2. 玻璃拟态卡片
```css
.glass-card {
  background: rgba(30, 27, 75, 0.6);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(123, 58, 237, 0.3);
  border-radius: 20px;
}
```
- 悬停效果：边框发光增强、轻微缩放

### 3. 按钮设计
- 渐变背景：蓝紫粉渐变
- 发光效果：外层光晕动画
- 悬停效果：缩放 1.05 倍，发光增强

### 4. 背景装饰
- 多层圆形光晕（模糊处理）
- 动态粒子效果（可选）
- 渐变线条装饰

### 5. 动效设计
| 元素 | 动画效果 | 时长 |
|------|----------|------|
| Logo | 浮动上下移动 | 6s |
| 卡片 | 悬停缩放 + 边框发光 | 0.3s |
| 按钮 | 脉冲发光 | 3s |
| 页面 | 加载淡入 | 0.5s |
